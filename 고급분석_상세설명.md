# 조직효과성 리포트 시스템 - 고급분석 상세 설명서

## 📋 고급분석 개요

### 고급분석이란?
조직효과성 진단의 **회사단위 리포트**에서만 제공되는 심화 분석 기능으로, 단순한 점수 분석을 넘어서 **조직의 정성적 특성과 잠재력을 과학적으로 분석**하는 고도화된 분석 도구입니다.

### 언제 사용되는가?
- **회사 전체** 또는 **대규모 조직** (응답자 5명 이상)의 진단 시
- **여러 팀/부서**가 포함된 종합적 조직 분석이 필요할 때
- **경영진 보고용** 전략적 인사이트가 필요한 경우

### 고급분석의 3가지 핵심 영역
1. **측면별 감성분석** (Aspect-based Sentiment Analysis)
2. **근거 기반 제안** (Evidence-based Recommendations)
3. **팀별 잠재유형 지도** (Team Potential Type Mapping)

---

## 📊 Page 1: 측면별 감성분석 (Aspect-based Sentiment Analysis)

### 🎯 이 분석의 목적
조직구성원들의 **주관식 응답을 5개 핵심 영역으로 분류**하고, 각 영역에 대한 **감정의 방향성과 강도**를 정량적으로 측정합니다.

### 📈 분석 방법론

#### 1단계: 자동 분류 시스템
주관식 응답을 키워드 기반으로 5개 영역에 자동 분류:

```
🏢 조직문화
├── 키워드: 문화, 분위기, 가치, 비전, 미션, 조직, 환경
├── 분석 대상: "회사 문화가 좋다", "조직 분위기가 경직되어 있다" 등
└── 의미: 조직의 무형적 자산과 구성원 인식

👨‍💼 리더십
├── 키워드: 리더, 상사, 관리, 지시, 의사결정, 방향성
├── 분석 대상: "상사의 지원이 부족하다", "리더십이 훌륭하다" 등
└── 의미: 관리자 역량과 리더십 스타일에 대한 평가

💼 업무환경
├── 키워드: 업무, 시설, 시스템, 도구, 환경, 근무
├── 분석 대상: "업무 도구가 노후되었다", "근무환경이 쾌적하다" 등
└── 의미: 물리적/시스템적 업무 조건

📚 성장기회
├── 키워드: 성장, 교육, 학습, 발전, 승진, 기회
├── 분석 대상: "교육 기회가 부족하다", "성장 가능성이 보인다" 등
└── 의미: 개인 발전과 경력 개발 기회

🤝 소통협력
├── 키워드: 소통, 협력, 팀워크, 커뮤니케이션, 협업
├── 분석 대상: "팀 간 소통이 원활하다", "협업이 어렵다" 등
└── 의미: 조직 내 상호작용과 협력 수준
```

#### 2단계: 감성 점수 계산
각 응답에 대해 긍정/부정 강도를 수치화:

```python
감성 점수 = 긍정 키워드 수 - 부정 키워드 수

긍정 키워드: 좋, 우수, 훌륭, 만족, 긍정, 효과, 성공
부정 키워드: 부족, 문제, 어려, 힘들, 부정, 실패, 개선

예시:
"리더십이 훌륭하고 만족스럽다" → +2점 (긍정)
"상사와의 소통이 부족하고 어렵다" → -2점 (부정)
"업무 환경은 보통이다" → 0점 (중립)
```

### 📊 결과 해석 가이드

#### 감성 점수 해석
- **+1.0 이상**: 매우 긍정적 → 조직의 강점 영역
- **+0.3 ~ +0.9**: 긍정적 → 지속 관리 필요
- **-0.3 ~ +0.3**: 중립적 → 개선 여지 존재
- **-0.9 ~ -0.3**: 부정적 → 개선 필요
- **-1.0 이하**: 매우 부정적 → 즉시 개선 필요

#### 실무 활용 예시
```
분석 결과:
조직문화: +1.2 (매우 긍정, 응답 15개)
리더십: -0.8 (부정적, 응답 12개)
업무환경: +0.1 (중립, 응답 8개)
성장기회: -1.5 (매우 부정, 응답 20개)
소통협력: +0.6 (긍정, 응답 10개)

📋 액션 플랜:
1. 성장기회 (최우선): 교육 프로그램 확충, 승진 체계 개선
2. 리더십 (2순위): 관리자 리더십 교육, 1:1 면담 강화
3. 업무환경 (3순위): 시설 개선, 업무 도구 업그레이드
```

## 💡 Page 2: 근거 기반 제안 (Evidence-based Recommendations)

### 🎯 이 분석의 목적
조직의 **강점은 더욱 활용하고, 약점은 체계적으로 개선**할 수 있도록 **데이터에 근거한 구체적인 실행 방안**을 제시합니다.

### 🔍 분석 방법론

#### 1단계: 강점/약점 자동 식별
주관식 응답을 긍정/부정으로 분류하여 패턴 분석:

```python
강점 식별 로직:
├── 긍정 키워드가 포함된 응답 수집
├── 빈도수가 높은 강점 영역 식별
├── 구체적인 강점 요소 추출
└── 활용 가능한 강점 자원 분석

약점 식별 로직:
├── 부정 키워드가 포함된 응답 수집
├── 심각도와 빈도를 기준으로 우선순위 설정
├── 근본 원인 분석
└── 개선 가능한 영역 도출
```

#### 2단계: 근거 기반 제안 생성
식별된 강점/약점을 바탕으로 실행 가능한 제안 도출:

### 📋 제안 유형별 상세 설명

#### 🌟 강점 활용 제안
**목적**: 조직의 기존 강점을 더욱 강화하고 확장 활용

```
강점 활용 제안 예시:

발견된 강점: "팀워크가 우수하고 협력적 문화가 잘 정착됨"

📈 제안 내용:
1. 모범 사례 전파
   - 우수 팀의 협력 방식을 전사 워크숍에서 공유
   - 팀워크 우수 사례 사내 게시판 게시

2. 강점 기반 확장
   - 크로스 펑셔널 프로젝트 확대 운영
   - 타 부서와의 협업 프로젝트 기획

3. 강점을 통한 약점 보완
   - 소통이 부족한 부서에 우수 팀 멘토링 지원
   - 협력 문화를 통한 개인 역량 개발 프로그램 운영

💼 실행 방안:
├── 담당: HR팀 + 우수 팀 리더
├── 기간: 3개월 (단기 실행)
├── 예산: 워크숍 운영비 + 인센티브
└── 성과지표: 다른 팀의 협력 점수 개선 정도
```

#### ⚠️ 개선 우선순위 제안
**목적**: 시급한 문제부터 체계적으로 해결

```
개선 우선순위 제안 예시:

발견된 약점: "교육 기회 부족, 승진 체계 불명확, 업무 과부하"

🔴 최우선 (즉시 개선): 업무 과부하
근거: 번아웃 위험, 이직률 증가 가능성
├── 업무량 재분배 및 우선순위 조정
├── 추가 인력 충원 검토
├── 업무 프로세스 효율화
└── 예상 기간: 1개월

🟡 2순위 (단기 개선): 승진 체계 불명확
근거: 동기부여 저하, 우수 인재 이탈 위험
├── 승진 기준 명문화 및 공개
├── 경력 개발 로드맵 제시
├── 정기적 경력 상담 실시
└── 예상 기간: 3개월

🟢 3순위 (중기 개선): 교육 기회 부족
근거: 장기적 역량 개발 필요
├── 연간 교육 계획 수립
├── 온라인 교육 플랫폼 도입
├── 외부 교육 지원 확대
└── 예상 기간: 6개월
```

### 🎯 실무 활용 가이드

#### 제안서 작성 시 활용 방법
1. **경영진 보고**: 우선순위별 개선 계획 수립
2. **예산 계획**: 개선 방안별 필요 자원 산정
3. **성과 관리**: 개선 효과 측정 지표 설정
4. **조직 개발**: 강점 기반 조직 역량 강화 전략

#### 의사결정 지원 정보
```
의사결정 매트릭스:
                시급성  중요성  실행난이도  예상효과
업무 과부하        높음    높음      쉬움      즉시
승진 체계         중간    높음      중간      3개월
교육 기회         낮음    중간      어려움    6개월
```

## 🗺️ Page 3: 팀별 잠재유형 지도 (Team Potential Type Mapping)

### 🎯 이 분석의 목적
**각 팀/부서의 고유한 특성과 잠재력**을 시각적으로 매핑하여, **맞춤형 조직 개발 전략** 수립을 지원합니다.

### 🔍 분석 방법론

#### 1단계: 팀별 데이터 수집
```python
팀별 분석 대상:
├── 부서명/팀명으로 그룹핑
├── 각 팀의 정량 점수 (NO1~NO39)
├── 각 팀의 주관식 응답 (NO40~NO43)
├── 팀별 응답자 수 및 참여율
└── 팀별 응답 패턴 분석
```

#### 2단계: 잠재유형 분류 시스템
각 팀을 4가지 잠재 유형으로 분류:

### 🌟 팀 잠재유형 4분류

#### 🚀 혁신 주도형 (Innovation Driver)
**특징**: 높은 성과 + 높은 만족도
```
식별 기준:
├── 정량 점수: 평균 4.0점 이상
├── 주관식 감성: 긍정적 (+0.5 이상)
├── 혁신/도전 키워드 빈출
└── 변화 수용성 높음

💪 강점:
- 자발적 개선 및 혁신 추진
- 높은 업무 몰입도와 성과
- 조직 변화의 선도 역할

📋 개발 방향:
- 혁신 프로젝트 리딩 기회 제공
- 타 팀 멘토링 역할 부여
- 새로운 도전 과제 할당
- 권한 확대 및 자율성 강화

🎯 활용 전략:
- 조직 변화 시 Change Agent 역할
- 모범 사례 전파의 핵심 주체
- 신규 사업/프로젝트 실험 대상
```

#### ⚡ 성장 잠재형 (High Potential)
**특징**: 중간 성과 + 높은 의욕
```
식별 기준:
├── 정량 점수: 3.0~4.0점
├── 주관식 감성: 긍정적 성장 의지 표현
├── 학습/발전 키워드 빈출
└── 개선 의지 강함

💪 강점:
- 높은 학습 의욕과 성장 의지
- 변화에 대한 수용성
- 개선 아이디어 적극 제시

📋 개발 방향:
- 체계적 역량 개발 프로그램 제공
- 멘토링 및 코칭 지원 강화
- 단계적 책임 확대
- 성공 경험 축적 기회 제공

🎯 활용 전략:
- 중장기 핵심 인재 풀 구성
- 순환 보직을 통한 경험 확대
- 외부 교육 및 개발 프로그램 우선 참여
```

#### 🛠️ 안정 기여형 (Stable Contributor)
**특징**: 안정적 성과 + 현상유지 선호
```
식별 기준:
├── 정량 점수: 3.0~3.8점 (안정적)
├── 주관식 감성: 중립적
├── 현재 상황 만족 표현
└── 급격한 변화 우려 표시

💪 강점:
- 일관되고 안정적인 업무 수행
- 조직의 기본 운영 기반 제공
- 높은 충성도와 소속감

📋 개발 방향:
- 전문성 심화 및 특화 역량 개발
- 점진적 변화 관리 지원
- 현재 역할에서의 만족도 제고
- 안정적 성과 인정 및 보상

🎯 활용 전략:
- 조직 안정성의 핵심 축
- 변화 과정에서 불안감 해소 대상
- 기존 업무 프로세스 최적화 주체
```

#### ⚠️ 지원 필요형 (Support Needed)
**특징**: 낮은 성과 + 높은 불만도
```
식별 기준:
├── 정량 점수: 3.0점 미만
├── 주관식 감성: 부정적 (-0.3 이하)
├── 불만/문제 키워드 빈출
└── 개선 요구 강함

💪 잠재력:
- 문제 해결 시 큰 개선 가능성
- 솔직한 피드백 제공 능력
- 조직 문제의 조기 경보 역할

📋 개발 방향:
- 근본 원인 분석 및 해결
- 집중적 지원 및 개입
- 단기 성공 경험 제공
- 정기적 모니터링

🎯 활용 전략:
- 우선적 개선 지원 대상
- 조직 문제 해결의 테스트베드
- 성공 사례 시 모델링 효과
```

### 📊 잠재유형 매핑 시각화

#### 2x2 매트릭스 표현
```
성과 높음 │ 🚀 혁신주도형    │ ⚡ 성장잠재형
         │ (활용 + 확산)   │ (육성 + 지원)
         │─────────────────│─────────────────
성과 낮음 │ 🛠️ 안정기여형    │ ⚠️ 지원필요형
         │ (유지 + 개선)   │ (집중 지원)
         └─────────────────┴─────────────────
         만족도 낮음                만족도 높음
```

### 🎯 실무 활용 전략

#### 조직 개발 로드맵 수립
```
단기 전략 (3개월):
├── 지원필요형: 집중 개입 및 문제 해결
├── 성장잠재형: 단기 성과 창출 기회 제공
├── 안정기여형: 현재 역할 안정화
└── 혁신주도형: 변화 리딩 역할 부여

중기 전략 (6개월):
├── 유형별 맞춤 개발 프로그램 운영
├── 팀 간 교류 및 벤치마킹 활성화
├── 성과 모니터링 및 피드백
└── 점진적 역할 확대

장기 전략 (1년):
├── 전체 팀의 상위 유형으로 이동 목표
```
