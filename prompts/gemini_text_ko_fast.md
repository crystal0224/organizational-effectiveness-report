# 조직 특성 요약 분석

주관식 응답을 바탕으로 조직의 특성을 4문단으로 요약하세요.

## 입력 데이터
- 조직 이미지: {{no40_text}}
- 강점: {{no41_text}}
- 개선점: {{no42_text}}
- 장애요인: {{no43_text}}
- 응답자: {{respondents}}명
- 조직: {{org_units}}

## 출력 형식
순수 텍스트 4문단으로 작성하세요:

1. **조직 이미지**: 구성원들이 인식하는 조직의 전반적 특성
2. **주요 강점**: 조직이 가진 우수한 역량과 자원
3. **개선 필요점**: 보완이 필요한 영역과 구체적 이슈
4. **종합 진단**: 전체적인 조직 상태와 개선 방향

각 문단은 100-150단어로 작성하고, 객관적이고 구체적으로 서술하세요.
